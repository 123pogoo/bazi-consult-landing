📖 快速部署指南

1

下載 HTML 文件

從上方「頁面預覽」標籤下載 bazi-landing-page.html 文件。

2

部署到 Vercel（推薦）

最簡單的部署方式：

  • 在 GitHub 上創建新倉庫
  • 上傳 bazi-landing-page.html 文件
  • 訪問 vercel.com，連接 GitHub 倉庫
  • Vercel 會自動部署，獲得免費 URL
3

替換 Line 連結

在代碼中搜尋所有 YOUR_LINE_ID 並替換為您的 Line 連結：

搜尋：https://line.me/R/ti/p/YOUR_LINE_ID
替換為：https://line.me/R/ti/p/@abc123xyz
或：https://line.me/R/oic/YOUR_OFFICIAL_ACCOUNT_ID
4

自訂文案內容

根據需要修改以下內容：

  • 大標題：搜尋「八字易經算命，一看就懂你的命運」
  • 副標：搜尋「專業老師親自解析...」
  • 服務卡片：修改 .service-card 中的 h3 和 p 標籤
  • 信任元素：修改「已幫助超過 5000+ 人」等數字
  • 咨詢時間：修改「週一至週日 10:00 - 22:00」
5

替換或自訂圖片

方法 A（推薦）- 用 Canva 或 Midjourney 生成新圖片：

  • 使用代碼中的提示詞生成圖片
  • 上傳到 Imgur 或 Cloudinary
  • 在 CSS 中替換圖片 URL

方法 B（快速）- 修改現有圖片：

  • 直接修改 .hero-image-placeholder 中的 emoji 和文字
  • 或用 CSS 背景漸層替代
6

自訂顏色主題

在 CSS 中找到 :root 部分，修改顏色變數：

--primary-dark: #1a1a3e; /* 深藍 */
--accent-gold: #d4af37; /* 金色 */
--button-red: #e74c3c; /* 紅色 */
--button-green: #27ae60; /* 綠色 */
7

修改倒計時天數

在 JavaScript 部分找到：

endDate.setDate(endDate.getDate() + 7);

將 7 改為您想要的天數。

8

SEO 優化（可選）

修改頁面的 meta 標籤以改善搜尋排名：

  • <title>：頁面標題
  • <meta name="description">：頁面描述
  • <meta name="keywords">：關鍵詞
9
